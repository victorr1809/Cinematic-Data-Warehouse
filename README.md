# Cinematic Data Warehouse
Kho dữ liệu phim điện ảnh đa quốc gia từ năm 1990 tới hiện tại (có update) và Dashboard phân tích doanh thu, điểm IMDB, thể loại phim được yêu thích và những tác phẩm điện ảnh hay nhất từng được sản xuất
> Khi sẻ chia nỗi đau, đau buồn hay niềm hạnh phúc trên từng khung hình, khán giả sẽ thấy bớt cô đơn hơn với nỗi niềm của chính họ
